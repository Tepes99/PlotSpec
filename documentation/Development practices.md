
## Tech stack

1. Main language for project: Python
2. Package manager uv
3. Repository structure: Polylith for python monorepo
4. Plotly for the interactive plots
5. Configs in JSON format
6. Server: FastAPI
7. Front end: Vanilla HTML + CSS (Minimal/No JavaScript)
8. Data source DB: DuckDB
9. Deployment: Docker container and compose files
